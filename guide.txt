網頁設計期末專案：個人網站建置
一、專案主題
請運用本學期所學 HTML5、CSS3、Bootstrap 5 技術，設計並完成一個具有個人特色的「個人網站」。
網站內容可介紹自己、興趣、作品、學習歷程、未來規劃等，展現個人風格與網頁設計能力。
________________________________________
二、技術規範
僅能使用
•	HTML5 
•	CSS3 
•	Bootstrap 5 

三、網站架構要求
網站至少需包含以下 5 個頁面：
1. 首頁（Home）
內容需包含：
•	個人照片或代表圖片 
•	個人簡介 
•	網站歡迎訊息 
•	Bootstrap Hero Section （主視覺區塊）
________________________________________
2. 關於我（About Me）
內容需包含：
•	自我介紹
•	個人特質
•	興趣專長
•	學習經歷
________________________________________
3. 作品集（Portfolio）
至少展示：
•	3個以上作品 
每個作品需包含：
•	作品圖片 
•	作品名稱 
•	作品說明 
建議使用：
•	Bootstrap Card 
________________________________________
4. 技能介紹（Skills）
至少介紹：
•	HTML 
•	CSS 
•	Bootstrap 
需使用：
•	Progress Bar 
•	Card 
•	Grid System 
呈現技能熟練程度。
________________________________________
5. 聯絡我（Contact）
內容需包含：
•	姓名 
•	Email 
•	電話（可使用虛擬資料） 
•	聯絡表單設計 
※ 不需實際送出資料
________________________________________
四、版面設計要求
必須使用 Bootstrap 元件
至少使用下列 6 項：
□ Navbar
□ Carousel
□ Card
□ Button
□ Progress Bar
□ Form
□ Modal
□ Accordion
□ Alert
□ Badge
（至少選用 6 項）
________________________________________
必須使用 Bootstrap Grid System
網站需使用：
•	container 
•	row 
•	col 
進行響應式版面設計。
________________________________________
五、響應式網頁設計（RWD）
網站必須支援：
裝置	寬度
手機	<768px
平板	768px~991px
電腦	≥992px
要求：
•	不可出現水平捲軸 
•	圖片自動縮放 
•	導覽列正常運作 
•	內容排版不跑版 
________________________________________
六、CSS客製化要求
除了 Bootstrap 外，必須自行撰寫 CSS。
至少完成：
•	自訂配色 
•	Hover效果 
•	字體設定 
•	區塊陰影 
•	邊框效果 
CSS 程式碼至少 100 行以上。
________________________________________

學習目標
透過本專案，同學需展現：
•	HTML 網頁結構規劃能力 
•	CSS 視覺設計能力 
•	Bootstrap 元件應用能力 
•	響應式網頁設計（RWD）能力 
•	網站整體規劃與整合能力 
完成後應能獨立建置一個具有專業外觀的個人網站。
